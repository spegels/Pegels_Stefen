Basic Structure

HTML is a programming language used to create webpages. For HTML, the browser is the compiler that runs the text file.
HTML code includes text, links, and images outlined with tags. Tags, using <>, have an opening and closing, and format
the code with line breaks, new paragraphs, and headers.

In order to move the code down to a new line, "< br>" or "< p>" are some tags that will move text down a new line, since
HTML does not recognize tabs in the text file. All text and tags to be seen by the user is included in the "< body>" of the
text file. the "< head>", which comes before the body, includes hidden information that will not show up on the webpage.

To change the size and font of text, the "< strong>" tag boldens text, the "< em>" tag italicizes text, and "< h1>" and other header sizes
from 1-6 will create headers.

Links and Page Navigation

In order to add a link to an HTML document, the anchor tag must be used "< a>". Links are used whenever a website wants to allow
a user to access another page. To add a link, start the anchor, add "href" with the desired web address in quotes.

To add a page navigation, name a certain body of text by putting "name = ____" within your anchor, and then adding an href anchor statement
at the top of the page with a "#" before the name assigned below. This will allow the user to click on the navigation link and be directed
to the specified section of the HTML page.

Adding Images

To add an image to an HTML page, download the specified .jpg or .jpeg file, drag it into your HTML page's folder, and use the "< img>" tag
with "src = ___.jpg" to add the specified file. Height and width can also be edited, but this may distort the image. In the same img tag, set
height/width equal to a specified number, in quotes.

Tool Tips

To further explain links within an HTML document, tool tips are little pop-ups that show when hovering the mouse over a link or picture.
To add a tool tip, simply type "title = " to the text of your choice, within the first anchor of a link.
